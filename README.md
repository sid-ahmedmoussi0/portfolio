<h2>🛠️ Installation Steps</h2>

<p>1. Clone this repository</p>

```
`git clone https://github.com/sid-ahmedmoussi0/portfolio.git`
```
<p>2. Go into the repository</p>

```
`cd portfolio`
```
<p>3. Install dependencies</p>

```
`npm install`
```
<p>4. Start a local dev server</p>

```
`npm start`
```
